Definition of Cataholic:
”It’s like an alcoholic, but with cats.”

Idea :
Since I really like cat, I want to create an online cat related experience. This website is to test if you are really into cats or not. Simulating UI interface of Pinterest, this project will generate random cat images from API. Moreover, this project created a gamification feeling with many hidden triggers.

Tech behind it :
It created a drag/resizable feature allowing the user to create a customized user interface using vue-grid-layout.js. Moreover, using setTimeout, it created a movie scene feeling when the user clicked ”Hell No” button.

Features:
  • Drag/Resize
  • "Hellno" Trigger Animation
  • Hell Yes/Yes/No triggers with combo count
