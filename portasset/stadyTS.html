Prompt: To facilitate international students' online learning experience.

Stady Tutoring Session is an one-on-one online tutoring setting with features such as:
  • Text/Voice/Video chatting;
  • Screensharing;
  • Realtime drawing board; (Supporting tablets such as iPad and Surface)
  • Realtime code editor;
  • Realtime doc editor;
  • Documents viewer (supporting PPT, PDF ,Word, Xlsx file formats);
  • File Transfer;

Tech:
  • utilized WebRTC for video/audio/screensharing
  • utilized p5.js for drawpad
  • utilized ace.js for code editor
  • utilized webSocket for secure data transmission
  • The whole website utilized Polymer.js as framework
  • utilized Google API for doc viewer
  • Utilized AWS codebuild/codepipeline for auto deployment
