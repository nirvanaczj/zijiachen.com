Statement: Since when data privacy is something that we all want but can never obtain?

Idea: With inspiration from Windows 98, the last windows version that don't harvest your personal information, I built an in browser operating system - Ace98. In here, all data is stored in local storage within the browser , only utilized binary image for facial verification.

Tech:
  • Utilized localStorage for remembering user's selections. Including added elements, deleted elements, elements' positions, notebook content, choices of Fahrenheit or Celsius while displaying weather, last login time, etc.
  • Utilized Facebook's Azure face analyze/recognition API for facial analyze & recognition.
  • Modify default contextmenu & drag drop event listener to create an immersive in-browser opearting system experience.
  • Getting user vague geolocation using user's IP address and get weather information using openweather API.
  • Heavily used iFrame for in-browser experience such as embeded music player.
  • Mainly utilized JS & Jquery for effects/dom manipulation.
  • Modeled and animated a 3D motion graphics of falling ball in Maya and used that as Screensaver.
